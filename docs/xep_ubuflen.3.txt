xep_ubuflen(3)
==============

NAME
----
xep_ubuflen - return length of ep_ubuf


SYNOPSIS
--------
*#include <xio/endpoint.h>*

*int *xep_ubuflen (char '*ubuf');*


DESCRIPTION
-----------
Return the ubuf size allocated by xep_allocubuf[3]

RETURN VALUE
------------
On success, length is returned.  On error, -1 is returned,
and errno is set appropriately.


ERRORS
------
*EINVAL*::
ubuf is not a valid endpoint's user-space buf

EXAMPLE
-------

----
char *ubuf = xep_allocubuf (0, 12);
assert (xep_ubuflen(ubuf) == 12)
----
