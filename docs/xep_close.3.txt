xep_close(3)
============

NAME
----
xep_close - close an endpoint


SYNOPSIS
--------
*#include <xio/endpoint.h>*

*void xclose (int 'eid');*


DESCRIPTION
-----------
Close the endpoint 'eid'.

RETURN VALUE
------------
Nothing returned for this function.

ERRORS
------
No errors for this function.

EXAMPLE
-------

----
int eid = xep_open (XEP_PRODUCER);
xep_close (eid);
----
