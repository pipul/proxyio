xbind(3)
==========

NAME
----
xbind - add an endpoint to the socket, local or remote


SYNOPSIS
--------
*#include <xio/sock.h>*

*int xbind (int 's', const char '*addr');*


DESCRIPTION
-----------
Adds a endpoint to the xsocket 's'. if the xsocket's type is XLISTENER, the
endpoint can be then used by other applications to connect to. or if the
xsocket's type is XCONNECTOR, the endpoint can be then used by caller connect
to remote listend xsocket.

The 'addr' is specific to the underlying transport protocol.

For the list of available transport protocols check the list on linkxio:xio[7]
manual page.

Maximum length of the 'addr' parameter is specified by XSOCKADDRLEN defined in
'<xio/sock.h>' header file.

RETURN VALUE
------------
On success, zero is returned.  On error, -1 is returned, and errno is set
appropriately.

ERRORS
------
*EBADF*::
The provided socket is invalid.
*EINVAL*::
The syntax of the supplied address is invalid.
*ENODEV*::
Address specifies a nonexistent interface.
*EADDRINUSE*::
The requested local endpoint is already in use.

EXAMPLE
-------

----
s = xsocket (XPF_NET, XLISTENER);
assert (xbind (s, "192.168.1.2:8080") == 0);

same as:
s = xlisten(XPF_NET, "192.168.1.2:8080");


s = xsocket (XPF_NET, XCONNECTOR);
assert (xbind (s, "192.168.1.2:8080") == 0);

same as:
s = xconnect(XPF_NET, "192.168.1.2:8080");

----
