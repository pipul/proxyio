xio(7)
======

NAME
----
xio - proxyio. Message Transmission Control Protocol


SYNOPSIS
--------
cc ['flags'] 'files' *-lxio* ['libraries']


DESCRIPTION
-----------

Following transport mechanisms are provided by xio socket:

In-process transport::
    linkxio:xpf_inproc[7]

Inter-process transport::
    linkxio:xpf_ipc[7]

TCP transport::
    linkxio:xpf_tcp[7]

Following APIs are exported by xio xsocket:

Create an xio socket::
    linkxio:xsocket[3]

Close an xio socket::
    linkxio:xclose[3]

Set a xio socket option::
    linkxio:xsetsockopt[3]

Get a xio socket option::
    linkxio:xgetsockopt[3]

Add local endpoint or remote endpoint to the xio socket::
    linkxio:xbind[3]

Add a local endpoint to the xio socket::
    linkxio:xlisten[3]

Add a remote endpoint to the xio socket::
    linkxio:xconnect[3]
    
Send a message::
    linkxio:xsend[3]

Receive a message::
    linkxio:xrecv[3]

Allocate a xmsg::
    linkxio:xallocmsg[3]

Deallocate a xmsg::
    linkxio:xfreemsg[3]

Get xmsg length::
    linkxio:xmsglen[3]
