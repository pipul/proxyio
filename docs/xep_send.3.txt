xep_send(3)
===========

NAME
----
xep_send - send a ubuf into endpoint


SYNOPSIS
--------
*#include <xio/endpoint.h>*

*int xep_send (int 'eid', char '*ubuf');*

DESCRIPTION
-----------
The function will send a ubuf containing the data from buffer pointed to
by 'ubuf' parameter to the endpoint 'eid'. the ubuf must alloc by
xep_allocubuf[3]

RETURN VALUE
------------
On success, zero is returned.  On error, -1 is returned, and errno is set
appropriately.


ERRORS
------
*EAGAIN*::
Non-blocking mode was requested and there's no ubuf to receive at the moment.
