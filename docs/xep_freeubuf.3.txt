xep_freeubuf(3)
===============

NAME
----
xep_freeubuf - deallocate a endpoint user-space buf


SYNOPSIS
--------
*#include <xio/endpoint.h>*

*void xep_freeubuf (char '*ubuf');*


DESCRIPTION
-----------
Deallocates a ep_ubuf allocated using linkxio:xep_allocubuf[3] function or
received via linkxio:xep_recv[3] function.

RETURN VALUE
------------
Nothing return

ERRORS
------

EXAMPLE
-------

----
char *ubuf;
assert( xep_recv (eid, &ubuf) == 0);
xep_freeubuf (ubuf);
----
